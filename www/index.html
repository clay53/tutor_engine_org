Tutor Engine

The open education suite

Focus on what you want

When you set target media, like a book, class, test, song, movie, or tv show, Tutor Engine will make sure that you learn those skills first.

Get immersed in your learning

As you learn a language, the interface will transform to be in that language.

Minimize repitition

The flashcards test multiple skills at once and allow you to set custom intervals

Seamless collaboration

With the GUID (Globally Unique IDentifier) Authority Server (GAS), new items (skills/media) can be identified without fear of conflicting with another user or data repository.

Learn online or offline

Once a copy of the app and repository is secured, non-online-multiplayer features of Tutor Engine can be used completely offline.

Release Timeline

After 4 years of development, the app became usable on Tuesday, November 14th, 2023.

The app is expected to go live with basic features for reading comprehension by December 25th, 2023. It will not be open source at this time.

The app's method will hopefully undergo formal experiments to measure its effectiveness to be completed by the end of May 2024.

A foundation will be created and funding will be secured to manage the development of the app summer of 2024.

The foundation will work on creating a custom software license or identifying an existing one to restrict competing commericial derivatives. The model is inspired by what is currently being done with Grayjay (an incredible app btw).

The source code for the app will become open source. Distribution platforms will only be able to make changes to the app that enhance/fix areas such as compatibility, security, and accessibility (anything can be done internally: such as personal use or within a school, but all of the source code must be made available to users).
